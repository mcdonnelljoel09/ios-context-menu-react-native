## 1. Installation

```sh
# install via NPM
npm install react-native-ios-context-menu

# or install via yarn
yarn add react-native-ios-context-menu

# then run pod install (uses autolinking)
cd ios && pod install
```

<br>

## 2. Usage

```js
import { ContextMenuView } from "react-native-ios-context-menu";
```
